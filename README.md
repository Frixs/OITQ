# One in the Quiver (REMASTER)

###The game in a nutshell

What is going on? It is a fighter minigame – deathmatch. Goal of this game is to reach required number of kills. You have only a couple of golds and your deadly shuriken at the start.\n\nIt is not simple task to survive… Can you beat it?
